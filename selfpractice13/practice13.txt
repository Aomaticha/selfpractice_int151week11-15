ต่อมาข้อ 2 จะเป็นแบบข้อ 1 เลยแต่เป็นเรื่อง การจัดการหนัง movie manager
ข้อนี้ก็ได้ลองเขียนได้ลองฝึกเขียนมากขึ้นทำให้เขียนได้คล่องมากขึ้น  และได้ลองเขียน async function มากขึ้น
ได้ฝึกเขียน  Fetch กับ HTTP Method ต่าง ๆ
ใช้ GET — ดึงข้อมูลหนังทั้งหมด
const res = await fetch("http://localhost:3000/movies");
const movies = await res.json();
ใช้ POST — เพิ่มหนังใหม่
fetch("http://localhost:3000/movies", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ title: movieTitle })
});
ใช้ PUT — อัปเดตชื่อหนัง
fetch(`http://localhost:3000/movies/${id}`, {
  method: "PUT",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ title: newName })
});
และใช้ DELETE — ลบหนัง
fetch(`http://localhost:3000/movies/${id}`, {
  method: "DELETE"
});
สรุปความรู้ที่ได้จากข้อนี้ ได้รู้วิธีดึงข้อมูลจาก API  รู้การส่งข้อมูลแบบ JSON และได้ใช้ fetch ครบทุก method (GET/POST/PUT/DELETE) สุดท้ายได้รู้ว่า  frontend คุยกับ backend ยังไง
